Definindo Variáveis CSS

As variáveis de CSS, que também são chamadas de propriedades customizadas, trazem grandes benefícios e vão funcionar de forma bem semelhante às variáveis de linguagens de programação, ou seja, sua maior função é armazenar valores que serão utilizados posteriormente no código.

      Sua criação é bem simples e se dá dentro da pseudo-classe root.

    1. color : muda a cor do texto. ...
    2. font-family : define uma lista de fontes para ser utilizada no elemento (e seus herdeiros). ...
    3. font-size : muda o tamanho do texto.
    4. font-weight : muda o peso da fonte.
    5. font-style : muda o estilo da fonte.
