Hierarquia CSS

Estabelece uma prioridade para aplicação de uma regra de estilo, caso se tenha mas de uma regra para um mesmo seletor CSS.

    Se houver um conflito, o navegador irá:

  • Analisar todas as regras;
  • Ver o nível de importacia na origem do elemento;
  • Comparar a especificidade das declarações com mesmo nível de importancia (entre o cálculo de especifidde);
  • Em caso de empate do grau da especificidade, irá olhara ordem de declaração colocada dentro dos arquivos de codigo. A regra CSS for declarada por último,
